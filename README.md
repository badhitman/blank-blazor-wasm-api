# Конструктор
Инструмент создания/описания бизнес-моделей через WEB интерфейс.

Описанные модели выгружаются в готовые C# классы и базовый набор служб доступа к данным.

Сформированные файлы кода легко интегрируются в основную заготовку (ветка main) таким образом, что на выходе получается готовый контекст BD с минимальным набором служб.
